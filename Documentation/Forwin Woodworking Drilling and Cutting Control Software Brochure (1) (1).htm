1.1.1 Coordinate column

l  Mechine: the values displayed for each axis are values calculated using the mechanical zero point as a reference point. Because the mechanical zero point is uniquely fixed, only axis movement can change the machine coordinates;

l  Workpiece : the values displayed for each axis are those calculated using the current system zero offset as a reference point. Even if there is no movement in the machine axes and the zero offset is changed, the values of the three axes will change accordingly;

1.1.2 Manual movement

Manual column is mainly to control the feed movement of each axis, mainly continuous, handwheel, incremental, the percentage of the multiplier can be controlled by the speed of the current feed multiplier display

As shown in Figure 1-3:

 

(Figures 1-3)

 

 

Ø  Continuous feed:When switching to manual window, the system defaults to manual continuous feed mode, click <Continue> button in the table control bar, you can make the table move continuously along the selected direction of the selected axis.

 

Ø  Handwheel feed :Under the handwheel feed, the operator can make micro-movement by rotating the hand crank pulse generator on the handwheel. The minimum distance that the tool moves when the hand pulse generator rotates by one scale (can be set by parameter), the system default "×1, ×10, ×100" corresponds to the movement amount of "0.001, 0.01, 0.1" (unit in mm) respectively. Handwheel feed is mostly used for parts machining. The handwheel feed is mostly used for the adjustment of parts processing and the setting of the zero point of the processing workpiece.

Fast: fast operation in manual mode

Slow: slow operation in manual mode

Ø  Incremental Mode: When the button is selected, the axis movement will go the current incremental value by the value entered;

1.1.5 Setting parameters

 

  Delay parameter \ Anti-collision parameter:

 

 

 G0 speed: programme empty running speed, unit mm/min;

 Manual slow speed: Continuous mode low speed operation in mm/min;

 Manual High Speed: Continuous mode high speed operation in mm/min;

 spindle arrival delay: spindle start delay processing, unit s;

 front press delay: delay time after the output of pre-pressing material cylinder, unit s;

 back press delay: Post-compression material cylinder output delay time, unit s;

 drill press delay: the time delay for the drilling and pressing cylinders to move into position;

 Y avoidance (U axis): Maximum travel during Y-axis avoidance;

 U-axis anti-collision: If the U-axis does not exceed the "U-axis anti-collision", the Y-axis travelling is unlimited; if the U-axis exceeds the "U-axis anti-collision", the Y-axis can only go to the "Y avoidance (U axis)" position;

 drill down delay: The time when the drill lift cylinder action is completed;

 drill press limit(Y-): Negative limit of Y-axis movement for normal downward pressure of the drill press cylinder.

 drill press limit (Y+): Positive limit of Y-axis movement for normal downward pressure of the drill press cylinder.

 drill push limit(Y-): the drilling top plate works normally and the Y-axis moves the negative limit.

 drill push limit(Y+): The drilling top plate is working normally and the Y-axis movement is positively limited.

 front press limit (Y-): The front pressure works normally and the Y-axis moves to the negative limit

 front press limit (Y+): The front pressure is working normally and the Y-axis movement is positively limited.

 front press limit (X-): The front pressure works normally and the X-axis moves negative limit.

 front press limit (X+): The front pressure is working normally and the X-axis movement is positively limited.

 

 

  Positional parameters:

 

 

 

End the meeting point (ticked):

(1) No, processing is completed and stops at the current position; (2) Yes, processing is completed and returns to the loading point;

 

Loose end manterial (ticked):

(1) No, end no loose ends; (2) Yes, end loose ends; (3) Signal, end wait to continue running signal loose ends

 

  X-axis load point: The programme activates the safety bit of the X-run, in conjunction with the gantry loading, to avoid touching;

 

  Y-axis load point: The programme activates the safety bit of the Y-run to match the gantry loading to avoid collision;

 

  U-axis load point: Side push cylinder output position after Y-axis moves into place;

 

  end material X pos: After all whole material movements have been travelled, Y moves into position and outputs the press wheel 1/2;

 

  end material Y pos: exceeding the set value requires resetting the pressure wheel cylinder to avoid collision, illustration (1-8)

 

  end material delay: Loose end material delayed back to fixed point;

 

  Drilling preparation X: When the drilling programme is executed, the X-axis runs to the ready position to start machining;

 

  Drilling preparation Y: When the drilling programme is executed, the X-axis runs to the ready position to start machining;

  Y-axis location: the marking position where the rapid feed arrives and starts to feed the material at the feed speed at this position;

  Load speed: The speed at which the feed shaft moves;

  Load offset: Compensation value for feeding;

1.7.3 Operational authority

Figure 1-31 Operating Privileges

 

The system has 3 levels of authority:

1>    Operator: Operator privileges do not require a password; the operator can perform some running operations;

2>    Debugger: debugging password 0000; the debugger can carry out process parameters (section 1.1.7) and set the zero point of the workpiece (section 1.1.5);

3>    Developer: Development code 4321; the developer can set the mechanical zero point (section 1.1.4), modify the system parameters (section 1.9.7), and the servo parameters (section 1.9.8).

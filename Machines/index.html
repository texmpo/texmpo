    Наименование файлов и комментарии

    УП группируются в папку с именем в формате (Вид Мебели / название / номер ).

    Пример:

    Имя файла должно содержать:

    • наименование детали
    • размеры заготовки
    • номер чертежа

    Для нестинга:

    УП группируются в папку с именем в формате (Вид Мебели/название/номер).

    1. Имя файла должно содержать сокращенное :

      1. названия изделия
      2. наименование детали
      3. материал заготовки
      4. размер заготовки нестандартного размера
      5. номер чертежа

    2. Комментарий (первая строка УП) дублирует название УП
